# SQL Queries

This directory contains SQL queries used for data analysis and backtesting.

## Queries

### `top_tickers.sql`
- **Purpose:** Find the top 10 tickers with the most data in the `tiingo_prices` table.
- **Last updated:** 2025-05-20

### `commodity_tickers.sql`
- **Purpose:** Check for commodity tickers (gold, silver, copper, lithium, etc.) in the `tiingo_prices` table.
- **Last updated:** 2025-05-20
